• I changed the filename from AirRaceScript3 to AirRaceScript4  
• Added capability to have multiple laps.  
• Added capability to use a group timer, instead of individual timers. 
If selected, the timer starts for everyone when the first plane enters gate #001.  
• This new version can replace the script in legacy .miz files without affecting them. You will get the new options.  
The only difference now being that flying backwards through the gates is penalized differently, so just don't do it.  
This is because with the addition of laps, the gate number will drop back to #001 after you pass last gate #010, or wahtever it is.  
• If your legacy .miz has multiple laps by overlapping gate zones at each pylon, just leave it as is with the new default settings (laps = 0)
• Updated note about minimum MIST version requirement because the older version of MIST can cause a script error.
• Added some additional sound effects in the repo to give users more options. If they prefer these sounds,
they would need to find-and-replace the appropriate .ogg callouts in the script.
